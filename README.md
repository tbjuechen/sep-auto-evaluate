# 国科大SEP快速评价脚本

## 需知
为了方便使用，将验证码 API token 明文写在代码中，目前余额还可以使用 5000 次，希望大家合理使用，不要滥用。如果额度耗尽，脚本的验证码识别功能可能会失效。

## 功能特性
- **一键评价**：自动填充课程和教师的评价表单。
- **智能识别**：自动识别验证码，无需手动输入。
- **随机内容**：评价内容在一定范围内随机，避免重复。

## 免责声明
本脚本仅供学习交流使用，请勿用于非法用途。使用本脚本产生的任何后果由使用者自行承担。

## 使用方法
通过release中的油猴脚本安装本脚本，在页面的悬浮窗中点击一键评价按钮评价。

## 发布渠道
- **GitHub Release**: 每次推送 `v*` 标签时，GitHub Actions 会自动构建并发布新版本。
- **GreasyFork**: 既然有了 GitHub Release，可以在 GreasyFork 设置脚本同步，URL 填 GitHub Release 中 `sep-auto-evaluate.user.js` 的下载链接（Latest）。
